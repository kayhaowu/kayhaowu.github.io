img Element

在 HTML 內加入圖片，可使用下列語法，src內填入圖片之相對路徑或絕對路徑

在其中加入alt屬性，用於當網速太慢、圖片出錯或是瀏覽器禁用圖片，導致圖片失效的時候，alt內容可作為圖片的替代文字進行補救。

在img標籤中加入width或height屬性可以調整圖片的寬與高

a Element

標籤a用於超連結，href內填入超連結的網址，若填入#會刷新該頁面，並回到該頁面的最上端

在其中加入target屬性
