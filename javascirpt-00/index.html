發表於 2022-12-15 更新於 2022-12-16 分類於 JavaScirpt ， Note

開發環境安裝

  • 編寫任何程式之前，需要安裝並配置一個程式程式語言的編輯器 ( Integrated Develop Environment，簡稱 IDE )，IDE 是任何開發人員的主要工具，我們使用它來編輯程式，推薦使用Visual Studio ( 簡稱 VS Code ) 作為 IDE ，其官方下載位址如下：

    ​ Visual Studio

安裝主題

使用內建主題

  • 可設定 VS Code 中已包含之主題，請點選 [ 左下角之齒輪圖示 ] -> [ Color Theme ] -> 選擇主題，如下圖所示：

安裝主題插件

  • 可依照下圖所標示之步驟進行主題安裝，下圖所安裝之主題插件為 Monokai Pro，此主題為付費主題，會在使用幾天之後詢問是否購買此主題，若不購買一樣可以正常使用，也可以選擇付費購買支持創作者，設定流程與上述相同，請點選 [ 左下角之齒輪圖示 ] -> [ Color Theme ] -> 選擇主題

VS Code 設定

Files：Auto Save

  • 設定自動存檔選項為 onFocusChange，此設定之功用為當使用者離開編輯器或點選其他視窗時，會自動儲存當下所編輯之檔案，其操作說明如下圖所示：

Editor：Format On Save

  • 使用搜尋欄搜尋 Format On Save 並勾選其選項，此設定之功能表示儲存時自動格式化，如下圖所示：
