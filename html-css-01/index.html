什麼是 HTML

  • HTML（ Hyper Text Markup Language ）超文本標記語言是開發人員用來建構與描述網頁內容的標記語言，由一系列標籤元素組成。透過設計網頁的元素排列結構，並藉由瀏覽器將 html 程式呈現為網站。

你的第一個網頁

  • 已將開發之編輯器安裝並配置完畢後，接著進行第一個網頁的 html 程式撰寫，第一支程式不外乎都是從最經典的 Hello World 開始：

    1. 於桌面新增一專案資料夾並以 VS Code 開啟
    2. 在資料節內新增一檔案為 index.html 之檔案，記住副檔名須為 .html
    3. 於程式碼撰寫區域輸入 ! 後，按下 Tab 鍵或選取第一個選項，VS Code 將自動生成基本的網頁架構

  • 第一支程式使用之標籤介紹如下所示：

    1. <!DOCTYPE html> ：定義這個檔案是一個HTML5的文檔
    2. <html>： HTML 的根元素
    3. <head>：包含有關HTML的元信息
    4. <meta>：
      • meta charset=UTF-8：指定網頁所使用之編碼
      • meta http-equiv=X-UA-Compatible：設置IE兼容模式
      • meta name=viewport：手機行動版網頁螢幕資訊
    5. <title>：定義HTML的標題（ 顯示在瀏覽器標題欄或頁面的選項中 ）
    6. <body>：所有可見內容的容器，如：標題、段落、圖像、超連結、表格與列表等
    7. <h1>：heading 1 標題
    8. <p>：段落
